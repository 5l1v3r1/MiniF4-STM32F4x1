# HAL sample program HAL库示例程序
## 中文
* `01-GPIO` GPIO 测试，按下KEY，LED快闪，KEY未按下，LED慢闪
* `02-ADC_RTC_Test` ADC和RTC测试，使用USB CDC虚拟串口显示数据
* `03-CDC_Standalone` USB虚拟串口例程，PA9,PA10为串口
* `04-USBDisk+FATFS` 模拟U盘和FATFS文件系统例程，需要外接FLASH
